# 🎨 Colour Picker App
A SwiftUI app demonstrating how to use the native ColorPicker view to let users select and apply custom colors.

---

## 🤔 What this is
This project demonstrates how to integrate SwiftUI's built-in `ColorPicker` into an app. It shows how to bind the selected color to state, apply it to UI elements in real time, and store it for use across the app. A quick reference for adding color selection to any SwiftUI project.

## ✅ Why you'd use it
- **ColorPicker binding** — Shows how to bind `ColorPicker` to a `@State` `Color` variable and react to changes
- **Real-time color application** — Demonstrates applying the selected color to backgrounds, text, and shapes as the user picks
- **Color persistence** — Covers storing and restoring a selected color using UserDefaults or similar

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/ColourPickerApp.git
cd ColourPickerApp
```

### 2. Open in Xcode
Double-click `ColourPickerApp.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- No third-party dependencies required.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
